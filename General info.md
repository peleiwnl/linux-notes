
## Linux Commands

- [[ls]] - Lists contents of directories
- [[cd]] - Changes current directory
- [[pwd]] - Displays current directory path
- [[cp]], [[my]], [[rm]] - Used to copy, move, and delete files respectively
- [[free]] - Shows memory usage
- [[df]] - Checks disk space usage
- [[top]] - Monitors running processes
- [[mkdir]] - Make a directory
- 
- [[netstat]], [[ifconfig]], [[ping]] - Help monitor and troubleshooting network connections

## Distributions

1. Ubuntu - Beginner friendly, large community
2. Kali Linux - Cyber security tasks such as pen testing
3. Debian - Stable and used as a base for other distros
4. Fedora - Latest software and technologies
5. Arch linux - Fully customise the system
6. Cent OS - Stable, server environments
7. Mint - Similar to Windows

## Components of a Linux Distro

1. Kernel - Core of the OS, managing hardware resources, controlling system processes, handling memory management and drivers. It is the bridge between hardware and user apps
2. System Libraries and Utilities - Provide standard functions and APIs that applications use to interact with the kernel
3. User Interface - Two types:
	1. GUI - Visual environment for system interaction
	2. CLI - For more advanced users through terminal commands
4. Software packages - distros come preloaded with various packages for specific use cases, including web browsers, productivity tools etc.
5. Package managers - for updating and removing software, e.g. APT

## A few benefits of Linux Distros

- Free, open source
- Customizable
- Stable and reusable
