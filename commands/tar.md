## Tape Archive

A powerful tool for creating, viewing, extracting, and managing archive files. Allows the user to bundle multiple files and directories into a single archive while preserving permissions and directory structure. It can also compress arhives using gzip, bzip2, or xz.

- Used to create backups and archive files
- Supports compression using gzip
- Preserves file permissions and structure
- Works with single or multiple files and directories
