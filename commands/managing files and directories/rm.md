## Remove a directory

We can remove directories with contents in it using rm.

### Flags

#### Removing a directory which has contents in it

We can use 'rm' with recursive flags <span style="color:rgb(24, 205, 136)">'-r'</span> or <span style="color:rgb(24, 205, 136)">'-rf'</span>:

```
rm -rf directory_name
```
