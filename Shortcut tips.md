
## ~ (Tilde)

A shortcut to the home base, no matter where you are, it will always bring you to the home folder.

![[Tilde example.png]]

## . (dot)

Simply means the folder you're in right now

## .. (Double dot)

one level above the folder you're in. If you're deep inside folders and want to go back a step, it will take you to the bigger folder that contains the one you are in.

```
cd ..
```

![[cd .. example.png]]

## tree: seeing the bigger picture in Linux

This command helps you to see all the folders at once, showing how al the folders are connected, like a family tree, so you can understand the bigger picture.

![[tree example.png]]
