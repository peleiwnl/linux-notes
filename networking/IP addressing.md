An IP address is a unique identifier assigned to devices on a network to enable communication, consisting of the network and host address. 

## What is IP Addressing?

Each device using a network receives an IP address and special identifier number. They are necessary for routing data packets between devices and enabling internet communication between devices. Dotted decimal notation, depicting for sets of bit. The network, separated by periods, is the most common way to express an IP address, which is 32 bits, and every number corresponds to a byte of the address.

There are two primary forms of IP address, IPv4 and IPv6. 

### Advantages of IP addressing

- Generate a unique ID number with the help of IP addressing for each device
- Necessary between different networks for performing data routing
- Can access device, servers, the internet, etc.

### Disadvantages of IP addressing

- Only limited number of IPv4 addresses
- Complexity in configuring the IP addresses
- If exposed, there is a very high chance of security threats.
