
## Creating a Script File

```bash
gedit hello.sh     # Creates a file named "hello.sh"
```

## Add Code

```bash
#!/bin/bash # Tells the computer to use bash
echo "Hello, World!" # Prints "Hello World"
```

## Make it Executable

Save the file, then type:
```bash
chmod +x hello.sh # GIve permission to ru nthe script
```

## Run it:

```bash
./hello.sh
```
